# Contribution Guide
## Aturan Branch
Gunakan format nama branch:
feature/nama-fitur-nama-mahasiswa
Contoh:
- feature/add-task-rina
- feature/delete-task-budi
- feature/update-status-sari
## Aturan Commit Message
Gunakan format:
tipe: deskripsi singkat
Contoh:
- feat: add task feature
- fix: fix status validation
- docs: update README
- refactor: simplify task search function
## Aturan Pull Request
Setiap Pull Request wajib berisi:
1. Deskripsi perubahan.
2. Fitur yang dikerjakan.
3. Cara testing.
4. Screenshot atau contoh output jika ada.
5. Minimal 1 komentar review dari anggota tim.
